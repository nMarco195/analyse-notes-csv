# Analyse des Notes de Deux Filières en Informatique

## Description
Ce projet est un mini-projet Python pour analyser les notes de deux classes en informatique.  
Il calcule :
- La moyenne des DS (DS1, DS2, DS3)
- La moyenne finale avec la pondération DS * 75% + Projet * 25%
- Le meilleur étudiant de chaque classe
- La moyenne par sexe (M/F)
- Comparaison de la performance entre les deux classes

## Contenu
- `main.py` : Script principal pour calculer les moyennes et analyser les résultats.
- `data/classe_A.csv` : Données de la Classe A
- `data/classe_B.csv` : Données de la Classe B

## Visualisation
- Histogramme de la distribution des moyennes finales
- Scatter plot : Projet vs Somme des DS
- Line plot : Moyenne finale par étudiant
